A simple and clean ASP.NET Core Web API built for learning, practice, and portfolio demonstration.  
This project includes basic CRUD operations and follows standard REST API practices.

---

 Features

- Create, Read, Update, Delete (CRUD) operations for Products  
- Entity Framework Core for database operations  
- SQL Server database  
- Swagger UI for testing APIs  
- Clean and easy-to-understand folder structure  

---

 Tech Stack

- ASP.NET Core Web API  
- C#  
- Entity Framework Core  
- SQL Server  
- Swagger / Swashbuckle
